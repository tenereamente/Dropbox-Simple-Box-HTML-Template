Skin/Colors How to change layout skin and colors ?



Light Layout

Place light_skin.css in all of the html pages right after main.css.
light_skin.css could be found inside assets/css/light_skin.css, For ease this file has already been placed in all files, you just need to uncomment that file




Colors

Premade colr style sheets could be found inside assets/css/colors/
Chose desired colr and place it after main.css

Unsticky Navigation

  • Find <nav> Tag in the header of html files
  • Add class "nav-stop" in the <nav> tag

Contact Contact form and google map

Google Map Address

It is easy to use google map of simlebox, In the HTML , find contact section and uou will see data attributes inside div that is responsible to create google map instance. You have to add yu own address and set desired zoom level: I.e. data-address="your address" data-zoomlvl="12"

Changin the email address in contact or newsletter form

open up the file php/contact.php , and change the following code:

$your_email="your_email@site.com";
